## Vireo

## Installing Dependencies

Type
<pre lang="sh">
npm install
</pre>
to install dependencies for the code

## Running

Run
<pre lang="sh">
grunt
</pre>
to start the server and then open the webpage at [http://localhost:3000](http://localhost:3000).
